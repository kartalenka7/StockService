## API для управления товарами на складах


### Команды Makefile

Создание и запуск контейнеров, необходимых для работы приложения 
````
make up
````

Остановка и удаление контейнеров
````
make down
````

Запуск тестов
````
make test
````

Запустить миграцию

````
migrate-up:
````	

Откатить миграцию

````
migrate-down:
````
